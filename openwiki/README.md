# Göteborgs-simulatorn 3D — Wiki

Agent-facing documentation for this repository. Start here, then jump to the
page that matches your task. Pages are intentionally concise and kept accurate
to the actual code.

> **What this is:** a browser-based 3D open-world sandbox of central Gothenburg,
> built from **real OpenStreetMap data**. You walk the city, ride trams and
> ferries, enter bars/clubs, take photos at landmarks, shop, and rack up (or
> shed) a wanted level — all in real Gothenburg wall-clock time and live-ish
> weather.

## Quick facts

|                     |                                                                                                     |
| ------------------- | --------------------------------------------------------------------------------------------------- |
| **Stack**           | React 19 · React Three Fiber v9 · @react-three/drei v10 · three 0.185 · zustand v4 · Vite 5         |
| **Language**        | Super-strict TypeScript, type-checked by **`tsgo`** (TS 7 native preview), **not** `tsc`            |
| **Entry point**     | `src/main.tsx` → `src/app/App.tsx`                                                                  |
| **World data**      | `public/geo/gothenburg.json` (committed OSM snapshot, ~6.4 MB) generated by `scripts/fetch-osm.mjs` |
| **Dev server**      | `http://127.0.0.1:8100` (`npm run dev`)                                                             |
| **Pre-commit gate** | `npm run check` (typecheck → lint → build), zero errors **and** zero warnings                       |
| **Path alias**      | `@/*` → `src/*`                                                                                     |
